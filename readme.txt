It is done automatically with each order.= Recommendations =* Need help with your Shopp ecommerce store? You'll need an account with [Wufoo](http://wufoo.com).2. Ensure that your [Shopp](http://shopplugin.net) is activated and up-to-date.3. Download Shopp + Wufoo. Unzip the `shopp-wufoo` folder.4. Upload the `shopp-wufoo` folder to the `wp-content/plugins/` directory.5. Activate Shopp + Wufoo through the Plugins menu in the WordPress admin.6. Go to Shopp 101 &rarr; + Wufoo. 7. Enter your API Key, Subdomain, and Hash for Wufoo and click Save Settings.== Frequently Asked Questions === How do I get a Wufoo account? =Visit [wufoo.com](http://wufoo.com) to register for an account.= What is Shopp and how do I get it? =Shopp is an ecommerce plugin for WordPress.
